#### The Problem with Heuristics

This approach **guesses** what changed based on file presence. It can't distinguish:
- "File was deleted by agent" vs "File was never in fossil"
- "File is new" vs "File was just never tracked"

This leads to issues like:
- `.fslckout` accidentally copied to upstream
- `.gitignore` deleted because it's in git but not in fossil
- Files from upstream deleted because they were never in fossil
